RDFa endpoint demo

This is a demo endpoint provided by Semargl Project. It can be used to extract RDFa data from web pages both as a web service or as a standalone web application.

Supported features

  • Core 1.0 (REC), Lite 1.1 (REC) and Core 1.1 (REC)
  • XHTML+RDFa 1.1 (REC)
  • HTML+RDFa 1.1 (WD)
  • SVG Tiny 1.2 Metadata (REC)
  • Optional features: Processor Graph Reporting (REC), Vocabulary Expansion (REC), Role Attribute 1.0 (PR)

Direct access

Service can be used directly via GET requests to this endpoint. You can check W3C recommendation conformance at rdfa-info.
