Supported W3C specifications

RDF

  • RDF Semantics (REC)
  • RDF Test Cases (REC)

RDFa

  • Core 1.0 (REC), Lite 1.1 (REC) and Core 1.1 (REC)
  • XHTML+RDFa 1.1 (REC)
  • HTML+RDFa 1.1 (WD)
  • Optional features: Processor Graph Reporting (REC), Vocabulary Expansion (REC), Role Attribute 1.0 (PR)

RDFa conformance can be checked at rdfa-info.

JSON-LD

Works in JSON-LD-RDF Converter mode. At this moment only supports conversion to RDF.

  • JSON-LD 1.0 (WD)

Other

  • RDF/XML Syntax Specification (REC)
  • SVG Tiny 1.2 Metadata (REC)
  • Terse RDF Triple Language (SUBM)
  • N-Triples (IWD)
