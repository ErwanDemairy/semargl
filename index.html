About

Semargl is a modular framework for crawling linked data from structured documents. The main goal of project is to provide lightweight and performant tool without excess dependencies.

At this moment it offers high-performant streaming parsers for RDF/XML, RDFa, N-Triples, streaming serializer for Turtle and integration with Jena, Sesame and Clerezza.

Small memory footprint, and CPU requirements allow framework to be used by any application. It runs seamlessly on Android and GAE.

Supported W3C specifications

RDF

  • RDF Semantics (REC)
  • RDF Test Cases (REC)

RDFa

  • Core 1.0 (REC), Lite 1.1 (REC) and Core 1.1 (REC)
  • XHTML+RDFa 1.1 (REC)
  • HTML+RDFa 1.1 (WD)
  • Optional features: Processor Graph Reporting (REC), Vocabulary Expansion (REC), Role Attribute 1.0 (PR)

Other

  • RDF/XML Syntax Specification (REC)
  • SVG Tiny 1.2 Metadata (REC)
  • Terse RDF Triple Language (SUBM)
  • N-Triples (IWD)
